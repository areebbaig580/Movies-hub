# MovieHub

MovieHub is a modern movie and TV discovery application built with **React**, **Tailwind CSS**, and the **TMDB API**. Users can discover trending content, search movies and TV shows, explore genres, save favorites, and view detailed information including cast, streaming providers, recommendations, and more.

# Features

## Home Page

- Trending Movies
- Top Rated Movies
- Trending TV Shows
- Personalized movie recommendations based on previous searches
- Recommendations based on previously searched genres
- Responsive horizontal sliders
- Modern dark themed UI

## Search

- Search Movies
- Search TV Shows
- Instant search using TMDB API
- Dedicated search results page
- Responsive movie cards

## Categories

- Browse all movie genres
- Discover movies by genre
- Explore content category-wise
- Dynamic genre filtering
  
## Movie & TV Details

Each movie/show contains

- Poster
- Backdrop Image
- Title
- Overview
- Release Date
- Runtime
- Genres
- Rating
- Cast
- Watch Providers
- Similar Recommendations
- More Like This section

## Wishlist

- Add movies to wishlist
- Remove movies from wishlist
- Wishlist stored using Local Storage
- Persistent across browser refreshes

## Recommendation System

MovieHub provides personalized recommendations by using

- Last searched movie
- Last searched TV show
- Previously searched genres

making the homepage more personalized for returning users.

## Responsive Design

- Mobile First
- Tablet Optimized
- Desktop Optimized
- Fully Responsive Layout
- Smooth UI Components

## Footer

Includes

- Contact Information
- Portfolio Links
- GitHub
- Featured Projects
